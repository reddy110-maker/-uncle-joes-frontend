# -uncle-joes-frontend
Frontend 

The frontend for this project will be the customer facing website that interacts with API and presents the data in a clear and presentable way. The frontend will allows users to browse the meny, search for store locations, log into their rewards, view past orders, and more. Its main role is to take data retured by the API and turn it into an easy to navigate web experience that feels modern and useful to customers.
